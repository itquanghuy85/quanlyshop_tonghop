# Release Notes v11.0.1+190 — Google Play

## Tiếng Việt (vi)

Phiên bản 11.0.1 — Cập nhật giao diện & bảo mật

🎨 Giao diện mới
• Tab Nhân viên & Tài chính: Thiết kế card ngang gọn đẹp, hiển thị tối ưu trên mọi kích thước màn hình
• Grid 3 cột trên tablet/desktop — không còn card to chiếm nửa màn hình
• Icon app mới cho Android, iOS và Web

🔒 Bảo mật
• Bảo vệ thông tin cấu hình nhạy cảm
• Cải thiện xử lý dữ liệu web

⚡ Cải thiện
• Dữ liệu web hiển thị chính xác ngay khi đăng nhập
• Tự động xóa cache khi cập nhật phiên bản
• 45+ cải thiện responsive cho màn hình rộng

## English (en-US)

Version 11.0.1 — UI & Security Update

🎨 New Design
• Staff & Finance tabs: Compact horizontal card layout, optimized for all screen sizes

🔒 Security
• Protected sensitive configuration files
• Improved web data handling

⚡ Improvements
• Web data loads correctly on login
• Auto cache-busting on version updates
